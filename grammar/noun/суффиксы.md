Наиболее распространенные суффиксы производных имен существительных:

| Суффикс   | Примеры                  |
| --------- | ------------------------ |
| -**er**   | worker, writer           |
| -**ment** | deployment, government   |
| -**ness** | happiness, kindness      |
| -**ion**  | restriction, connection  |
| -**dom**  | freedom, wisdom          |
| -**hood** | childhood, neighbourhood |
| -**ship** | leadership, friendship   |
